Web App "LaTeX Task Manager"

"LaTeX Task Manager" - a bilingual (EN/SK) web app for streamlined educational workflows.

Teachers: Log in, switch languages via flags, upload .tex files for task generation, view/export student lists in CSV, access individual student info/tasks.

Students: Log in, generate tasks from selected files (with images), submit solutions via textarea, logout easily.

  • Multi-language support with flag selectors
  • Secure login and role-based features
  • PDF generation for user documentation

Powered by Php, Bootstrap for responsive UI, with jsPDF for docs export.

Restaurant Menu API

"Restaurant Menu API" - a simple RESTful PHP API for retrieving and managing daily menus across multiple restaurants.

Key endpoints:

  • GET: Fetch JSON menu data for all restaurants by day/items/prices
  • PUT/POST: Update specific item prices (zero-based index)
  • POST: Add new items to every day
  • DELETE: Remove entire restaurants

No authentication, with 400/404 error handling. Hosted on STU server

Web Game "Multiplayer Pong"

"Multiplayer Pong" - thrilling real-time WebSocket Pong for 2 players, inspired by classic arcade!

Enter lobby by name, see player count. Field auto-resizes: 2-player classic, 4-player arena!

Key features:

  • WASD/arrows paddle control—defend as ball accelerates
  • 3 misses = eliminated (wall bounces ball)
  • Live bounce counter & remaining lives per player
  • Last player standing wins—players can leave anytime
