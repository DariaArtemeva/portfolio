Python Automation "Vending Sales Monitor"

A Python automation script that monitors vending machine sales via a web dashboard (Selenium) and sends Telegram alerts when sales increase above a configured threshold.

  • Automated login + navigation to sales statistics page
  • Filters target machines by ID/prefix and parses table rows
  • Normalizes sales values (regex cleanup) and tracks previous state
  • Sends Telegram notifications when sales grow by N+ items
  • Runs in a loop with periodic refresh (near real-time monitoring)

Tech: Python, Selenium, Regex, Telegram Bot API
